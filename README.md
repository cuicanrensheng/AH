# 📺 TVLive 直播APP

一款专为 **Android TV / 电视盒子** 打造的轻量级 IPTV 直播应用，简洁、稳定、无广告，支持多种自定义配置。

---

## ✨ 核心功能

| 功能 | 说明 |
| :--- | :--- |
| 📺 直播源支持 | 支持标准 M3U/M3U8 格式，可自定义频道分组 |
| 📱 扫码配置 | 手机扫码一键导入直播源和 EPG，无需手动输入 |
| 🔄 频道切换 | 支持遥控器上下键、手势滑动切换频道 |
| ↩️ 反向切换 | 可设置频道切换方向，适配不同用户习惯 |
| 🖼️ 画面比例 | 支持 原始 / 全屏 / 填充 三种模式 |
| ⏰ 开机自启 | 电视开机自动进入直播，秒开秒播 |
| 🚀 自动更新 | 云端构建发布后，APP 内一键升级 |
| 📅 EPG 节目单 | 支持自定义 EPG 地址，查看节目预告 |

## 📥 安装与使用

### 1. 下载安装
- 前往 [Releases 页面](https://github.com/cuicanrensheng/1/releases)，下载最新版本的 `app-release.apk`
- 安装到你的 Android TV / 电视盒子（需开启「未知来源应用安装」权限）

### 2. 配置直播源
有两种方式添加直播源：

**方式一：扫码添加（推荐）**
1.  打开 APP，进入「设置」→「扫码设置」
2.  用手机扫码，在网页中输入你的 M3U 直播源地址和 EPG 地址
3.  提交后 APP 会自动保存配置，重启即可生效

**方式二：手动添加**
1.  进入「设置」→「自定义直播源」
2.  输入你的 M3U 地址，点击保存
3.  重启 APP 后，频道列表会自动加载

## 🔧 自动更新说明

本项目通过 **GitHub Actions** 实现全自动构建与发布：
- 每次向 `main` 分支推送代码，都会自动构建 APK 并发布 Release
- 同时自动生成 `update.json` 文件，APP 启动时会自动检测新版本
- 如需关闭自动更新，可在设置中手动关闭「自动更新」开关

## ❓ 常见问题

**Q：为什么扫码设置后频道没加载出来？**
A：请检查直播源地址是否有效，网络是否正常，然后重启 APP 即可。

**Q：遥控器上下键切换方向反了怎么办？**
A：进入设置，开启「频道反向切换」开关，即可恢复你习惯的操作方向。

**Q：自动更新下载后无法安装？**
A：请在设置中开启「安装未知来源应用」权限，或在弹出的安装界面中允许该操作。

**Q：直播卡顿/无法播放？**
A：可能是直播源失效或网络不稳定，建议更换其他稳定的 M3U 源。
