# LineageOS 18.1 for 360 N7 (n7)
GitHub Actions cloud build template.

## How to use
1. Fork this repo
2. Go to Actions → Enable workflows
3. Run workflow: **Build LineageOS 18.1 (360 N7)**
4. Download ZIP from Artifacts when done

## Device
- Codename: n7
- CPU: SDM660 (Snapdragon 660)
- Android 11 (LineageOS 18.1)

## Notes
- Bootloader **must be unlocked**
- Flash custom TWRP for n7
- Wipe data before flashing
